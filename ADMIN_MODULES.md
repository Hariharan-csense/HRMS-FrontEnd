## What Admin Can Do

### View (ನೋಡು)
- Can see all modules and their data

### Create (ರಚಿಸು)
- Can create new employees, leaves, expenses, assets, etc.

### Edit (ಸಂಪಾದಿಸು)
- Can modify any existing records

### Approve (ಅನುಮೋದಿಸು)
- Can approve all requests (leaves, expenses, etc.)

## Admin Special Permissions

### Organization Setup
- Configure company information
- Manage branches, departments, designations
- **Most important**: Configure roles and their module permissions

### Full Reporting
- Access to all reports and analytics
- Employee data across all departments

### All Approvals
- Can approve any leave, expense, or request
